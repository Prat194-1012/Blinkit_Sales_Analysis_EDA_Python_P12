# 📊 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a **Blinkit Grocery Sales dataset** to uncover insights about product sales, outlet performance, and customer preferences.

The analysis was performed using **Python, Pandas, and Matplotlib** to transform raw retail data into **meaningful business insights**.

The goal of this project is to demonstrate how a **Data Analyst can use data to support business decision-making.**

---

# 🎯 Business Problem

Blinkit (India’s quick commerce grocery platform) sells thousands of products through multiple outlets.

Businesses need answers to questions like:

* Which **products generate the most sales?**
* Which **outlet types perform best?**
* Does **fat content affect product demand?**
* How do **location and outlet size impact sales?**
* Which **outlets generate the highest revenue?**

This project answers these questions using **data-driven analysis.**

---

# 📂 Dataset Information

The dataset contains sales data for different Blinkit grocery products.

### Key Features

| Feature                   | Description              |
| ------------------------- | ------------------------ |
| Item Type                 | Category of product      |
| Item Fat Content          | Low Fat / Regular        |
| Item Sales                | Revenue generated        |
| Item Rating               | Customer rating          |
| Outlet Size               | Small / Medium / Large   |
| Outlet Location Type      | Tier 1 / Tier 2 / Tier 3 |
| Outlet Establishment Year | Year outlet started      |
| Outlet Type               | Grocery store type       |

# 📊 Exploratory Data Analysis

## 1️⃣ Total Sales by Fat Content

Analyzed how **Low Fat vs Regular products** contribute to overall sales.

## 2️⃣ Sales by Item Type

Identified the **top-performing product categories** generating the highest revenue.

## 3️⃣ Sales by Outlet Size

Compared sales across **Small, Medium, and Large outlets**.

## 4️⃣ Sales by Outlet Location

Analyzed how **Tier 1, Tier 2, and Tier 3 cities** perform.

## 5️⃣ Sales by Outlet Establishment Year

Studied how **older vs newer outlets perform in revenue generation.**

## 6️⃣ Sales by Outlet Type

Identified which **outlet types contribute most to Blinkit's revenue.**

# 💡 Key Business Insights
